這是一台專為 RimWorld 打造的「全自動 AI 翻譯與本地化生成引擎」。它不僅能自動比對缺失的文字，還能呼叫最先進的 AI 模型進行術語級別的翻譯，並為你自動生成一個乾淨、獨立的本地化模組包！

✨ 核心黑科技特點
​
🌍 原生多國語系支援 (Multi-Language UI) 【V4.4 新增】
模組自身的設定介面與數位戰情日誌已全面本地化！完美支援繁體中文、簡體中文、日本語、한국어，讓全世界玩家都能無障礙使用。
​
🧠 AI 智慧精準焊接 & 多國翻譯
內建 API 介面，支援接入 Google (Gemini), DeepSeek, OpenAI, Grok 等多種主流 AI 模型。自動抓取模組中缺失的 Keyed 與 DefInjected 進行精準翻譯，並且內建術語保護（如 {0} 參數防破壞）。
​
🔗 自訂 API 與本地模型支援 (Custom Base URL) 【V4.4 新增】
新增自訂 API 網址功能！除了雲端大模型，現在你更可以無縫接入 LM Studio、Ollama、vLLM 等本地部署模型！API Key 留空也能跑，享受完全免費且斷網可用的本地化翻譯！
​
🌐 全域翻譯記憶體池 (Global Translation Memory)
完美解決「主模組」與「漢化補丁」分離的痛點！系統會在啟動時收集你所有已安裝模組的繁/簡文字，建立一個巨大的全域共用池。遇到缺漏時，優先從池中抓取（並自動進行繁簡轉換），最大限度節省 AI 呼叫額度和時間！
​
☢️ Defs 深層 X 光掃描 (Deep XML X-Ray)
原作者偷懶沒寫 Languages 資料夾？沒關係！本模組內建 X 光探針，能強制破解深埋在 1.6/Defs/ 等各層級目錄下的 XML 代碼，硬生生把標題 (label) 和描述 (description) 拔出來翻譯！任何英文單字都無處可躲！
​
📦 無痛獨立輸出 (!Translation_AI_Pack)
翻譯完成後，系統會自動在你的 Mods 資料夾中生成一個名為 !Translation_AI_Pack 的獨立模組。絕對不會修改或破壞你原本的任何模組檔案！
​
💻 數位戰情監控面板
內建超酷炫的即時滾動日誌！全彩顯示掃描進度、缺漏數量、AI 焊接狀態與連線測試結果。並且在每次任務結束後，內建記憶體自動釋放機制，保證遊戲效能 0 負擔，絕對不卡頓！

​
🛠️ 使用方法

1.訂閱並啟用本模組（建議放在載入順序較後面的位置）。

2.進入遊戲的「選項」 -> 「模組設定」 -> 找到「全自動補位翻譯」。

3.選擇你的目標語系（繁體中文 / 簡體中文 / 日本語 / 한국어）。

4.填入你申請的 API Key，點擊「獲取模型」並選擇。（若使用本地大模型，請在自訂 Base URL 填入本地網址，Key 可留空）。

5.點擊「🚀 開始執行自動補位計畫」，看著全自動產線幫你把所有英文翻譯成完美的母語！

6.跑完後，記得在模組清單中啟用剛剛自動生成的 !Translation_AI_Pack 模組，並且務必排在所有模組的最下面！！！（這很重要，確保翻譯能正確覆蓋所有模組）。

​
⚠️ 注意事項

1.本工具為「按需執行」的腳本，只有在您點擊執行時才會運作，對遊戲平時運作的 TPS/FPS 影響為 0。

2.本模組採用非同步運作機制，當產線執行時，您可以關掉設定視窗繼續遊戲遊玩，隨時回來查看進度即可。

3.自動生成的翻譯包僅供個人本地使用，讓您享受最完整的 RimWorld 體驗！
